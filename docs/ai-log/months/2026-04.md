## 2026-04-16 | Saldo diario de horas por card

Status: approved
Kind: feature, backend, frontend

### What Changed
- Added backend helpers to calculate daily worked minutes from closed real punch pairs and compare them against an 8-hour target.
- Extended each day payload with balance fields so the initial page render and week navigation API return the same overtime/debt state.
- Added a balance row to each day card showing `Hora extra`, `Horas devendo`, or neutral day balance with dedicated styling.

### Why
- Each day card needed to show whether the user exceeded or fell short of the expected 8-hour workday based on real punch records.

### Files
- app.py
- templates/index.html
- static/style.css

### Verification
- `python -m py_compile app.py db.py scheduler.py punch.py`
- Manual helper scenarios checked positive, negative, half-day, and incomplete day balance outputs.

### Open Edges
- No browser-based visual validation was performed after the UI change.
- The balance is shown only for closed days with either `entrada` plus `saida` or the full `entrada/pausa/retorno/saida` sequence.
