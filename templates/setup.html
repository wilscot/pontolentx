Fluxo recomendado

Se o coletor já existe no Chrome, mantenha o Chrome como navegador operacional e importe a sessão direto dele. Se o coletor existe só no Edge, use a aba Edge para copiar a sessão primeiro; se a migração para o Chrome não reaproveitar o coletor corretamente, troque o navegador operacional para Edge e valide novamente.

Esse diretório dedicado do Chrome será usado quando o navegador operacional estiver em Chrome.
Copia cookies e Local Storage do Chrome pessoal para o perfil dedicado do navegador operacional escolhido abaixo. Feche o Chrome antes.
Como usar com Chrome Fluxo padrão para quem já tem o coletor e a sessão salvos no Chrome.
  1. Feche o Chrome pessoal completamente para liberar o arquivo de cookies.
  2. Clique em Importar sessão do Chrome para copiar cookies e Local Storage do perfil pessoal Default.
  3. Se precisar cadastrar o coletor do zero, clique em Abrir Chrome para configurar, faça login no Pontotel e registre o coletor informado.
  4. Salve as configurações, volte para a home e use Testar agora antes de deixar o agendamento rodando.
Esse diretório dedicado do Edge fica pronto para o fallback quando o Chrome não reaproveitar a sessão importada.
Copia cookies e Local Storage do Edge pessoal para o perfil dedicado do navegador operacional escolhido abaixo. Feche o Edge antes.
Como usar com Edge Fluxo pensado para quem já tem o coletor e a sessão ativos só no Edge.
  1. Deixe Chrome como navegador operacional na primeira tentativa para migrar a sessão do Edge e continuar batendo ponto pelo Chrome.
  2. Feche o Edge pessoal completamente e clique em Importar sessão do Edge. A cópia sai do perfil pessoal Default.
  3. Salve, volte para a home e valide com Testar agora. Se o Chrome não reaproveitar a sessão ou o coletor, volte aqui e troque o navegador operacional para Edge.
  4. No fallback com Edge, importe novamente ou abra o Edge dedicado, confirme login/coletor e rode um novo teste antes de ativar o agendamento.
